## 第一章

搭建 hexo 博客需要的几样东西：
1. 一台联网的电脑 2.git 3.nodejs 4. 脑子 5. 手（狗头保命
几个必须的较长的命令：
1.（安装国内的 cnpm (淘宝源)）npm install -g cnpm --registry=https://registry.npm.taobao.org
2.（安装 hexo 框架）cnpm install -g hexo-cli
3.（安装部署到远端的 hexo 工具）cnpm install --save hexo-deployer-git
注意事项：
1. 在部署到远端之前记得先本地生成预览一下
2. 部署到 GitHub 服务器之前记得往博客里面添加 GitHub 的账号信息
3. 如果部署到远端失败就尝试推倒墙娘然后再试一次。
4. 在 config 里面最下方的 repo 后面写 GitHub 给的链接，在 branch 后面填 master
